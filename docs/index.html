Introduction

One of the of advantage of coding is to be able to extract the exact information and predict natural phenomenon behaviors as well as the impact of human being on the environment in order to find the right solution at the right time. In that logic, R programing seem to be a suitable tool to propose a range of solutions to leaders who might decide what to do in any given situation in general but in the atmospheric behavior in particular which might slow not only the economy but also the conquest of new frontiers in space. In this project, we are interested in studying the chaos in the atmosphere because its behavior is extremely important for space exploration. In fact, any launch system built to send astronaut in space or deliver satelitte in space will take off if and only if the atmospheric conditions are favorable.

Materials and methods

I firstly install the differential equation package followed by the “Desolve” packages. I then used the Lorenz function to specify the parameters in X-Y-Z direction which is 3D. After that I created my parameters (a, b, and c) I will use as well as the sequence time before plotting my results.

Lorenz <- function(t, state, parameters) {
  with(as.list(c(state, parameters)), {
    dX <- a * X + Y * Z
    dY <- b * (Y - Z)
    dZ <- -X * Y + c * Y - Z
    list(c(dX, dY, dZ))
  })
}

parameters <- c(a = -8/3, b = -10, c = 28)
state <- c(X = 1, Y = 1, Z = 1)
times <- seq(0, 100, by = 0.01)
out <- ode(y = state, times = times, func = Lorenz, parms = parameters)
plot(out)
